# Valentino Juan | Portofolio Website

## Overview
This is my personal portofolio website built to showcase my journey as a Full-Stack Software Engineering student at RevoU.
It highlights my backgrounds, projects I've been working on, and contact section.

The main goal of this project is to present my professional identity and growth to become a Software Engineer.

## Features Implemented
### 1. Home Section
Section to introduce myself.
### 2. About Section
Section to describes more about me including my backgrounds, skills and interests.
### 3. Project Section
Section to show what project I've been working on.
### 4. Contact Section
Section to show my contacs or write a form to send me a message.
### 5. CSS
Styling the page to be responsive across all devices.


## Technologies Used
* HTML
* CSS
* Tailwind
* Google Fonts : DM Sans, Cormorant Garamond
